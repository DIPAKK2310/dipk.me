# Dipk.me 
## This is portfolio website with sleek design and modern animation

## Getting Started


### Installation

#### 1. Clone the repository

```bash
git clone https://github.com/DIPAKK2310/dipk.me.git

```

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

# Tech Stack
 - NextJs
 - Tailwindcss
 - radix-ui
 - ThreeJs
 - Motion
 - Lucid-react
 - Lenis
